## Tip

#### maven 命令

mvn -v 验证maven安装有效
mvn complie 生成.class文件包
mvn package 打包
mvn install 将code打包并添加到local仓储下面
mvn test 运行测试

#### linux 命令

free 查看linux服务器内容使用的情况
top 查看系统进行资源占用情况
